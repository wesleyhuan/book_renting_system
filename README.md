Rent Page:
1. Fields for Username, Book Name, and Rent Date.
2. Saves the data instantly.

Return Page:
1. You enter the Username.
2. It lists only the books currently rented (active).
3. You Select the books via checkboxes (checkboxes allow selecting multiple books at once).
4. When you click "Return Selected Books", it automatically removes them from this list and saves the current date as the return date.

My Books (Status) Page:
1. Enter Username.
2. It lists all details: Book Name, Rent Date, Username, and current status (whether it is returned or still rented).
